PeewPeew v1.0.2

Peewpeew is a 2D single player shooter game. It is in its early development so you might experience some bugs. The highest score is the winner, that's the main goal of this game for now. It doesn't have various gamemodes and levels. I will continue to add features to this web game so it can be more fun and have a good user experience.

Controls

Move : (WASD controls) [W = UP] [A = RIGHT] [S = DOWN] [D = LEFT] / (MOUSE controls) LMB

Shoot : (WASD controls) LMB / (MOUSE controls) SPACE

Reload : R

Change Firing Mode : Q
